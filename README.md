- SELECT * FROM `students` WHERE `date_of_birth` LIKE '1990%';

### SELEZIONARE TUTTI GLI STUDENTI CHE HANNO PIU' DI 30 ANNI
- SELECT * , TIMESTAMPDIFF(YEAR, `date_of_birth`,CURDATE()) AS `age` FROM `students`;
